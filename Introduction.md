# Games

This is a simple platformer games with 3 different stages, each stages require player to collect all collectibles (apples) and kill enemies to gain points. With enough points, player can touch the trophy at the end of each map to advance to the next level.

Player can press "Esc" to pause the game
User input :
  Q to dash forward
  Space while midair to jump (3 times max)
  Left_mouse_button to perform attack
